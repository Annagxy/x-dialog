x-dialog

reactjs的弹窗浮层组件

我认为，每一个组件都应该有他自带的样式和属性事件回调配置。所以我会给x-dialog默认一套简单的样式，和各种默认的配置项。

Install

使用 npm 安装, 运行

$ npm install x-dialog
